## 3. 数值逆运动学 (`numerical_ik_r7.cpp`)

**接口：**
```cpp
int numerical_ik(const double T_target[4][4],   // 目标位姿 (位置 mm)
                 const double q_init[7],         // 初始猜测 (度)
                 double q_out[7],                // 输出 (度)
                 int    max_iter = 200,
                 double pos_tol  = 0.001,        // 位置容差 mm
                 double rot_tol  = 0.00001,      // 姿态容差 rad
                 bool   verbose  = false,
                 const double* q_prefer = nullptr);  // 零空间偏好配置
```

**算法流程：**

```
1. 初始化 q = q_init
2. 迭代循环：
   a. FK 计算当前位姿 T_cur
   b. 误差 e = [Δp (mm), Δr (轴角·rad)]，缩放 Δp *= 1/100
   c. 收敛判断 (pos < 0.001mm, rot < 1e-5 rad)
   d. 数值雅可比 J (6×7)：扰动 dq=1e-4 rad，FK 差分
   e. 主任务解：dq = Jᵀ (JJᵀ + λ²I)⁻¹ e
   f. 零空间投影：dq += (I − J⁺J) · k · (q_prefer − q)
   g. 线搜索：步长 {1.0, 0.5, 0.25} 选最优
```

**关键设计决策：**

| 问题 | 方案 |
|------|------|
| J 行量级不匹配（mm vs rad） | 位置行缩放 1/100 |
| 奇异位姿 | 阻尼最小二乘 λ=0.1 |
| 7-DOF 冗余多解 | 零空间投影向 q_prefer 靠拢 |
| 过冲/震荡 | 线搜索选最优步长 |
| 数值扰动太小被截断 | dq_rad=1e-4 转 dq_deg≈0.0057° |

**性能：** 收敛 3~7 次迭代，耗时 28~48 μs

**编译：**
- 独立运行：`g++ numerical_ik_r7.cpp r7_fk_lib.cpp`
- 供其他文件调用：加 `-DIK_LIB_ONLY` 排除 main()
