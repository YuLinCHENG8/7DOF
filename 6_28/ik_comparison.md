# 7-DOF 解析逆运动学算法对比

对比文件：`6_26/analytical_ik.cpp`（cpp版）vs `6_28/tsinghua_paper.py`（py版）

---

## 整体策略

| 维度 | `tsinghua_paper.py` | `analytical_ik.cpp` |
|---|---|---|
| 参数化方式 | 几何法：先算肘关节**位置 E**，再从 E 反解角度 | 矩阵法：直接参数化 **R_0_3 = A·sinψ + B·cosψ + C**，再做 ZYZ 分解 |
| 核心数学工具 | 圆几何 + 直接 FK 位置方程 | Rodrigues 旋转公式 |
| 适用场景 | 离线规划、初始化 | 实时控制（500 Hz） |

---

## py 版求解流程（几何路线）

### Step 1：确定肩心 S 和腕心 W

```
S = (0, 0, d1)                          # 肩心，固定在基座上方
W = p_target - d8·z7 - d6·z7           # 从末端沿 z7 轴退两段得腕心
u_sw = normalize(W - S)                 # 肩→腕单位向量
```

### Step 2：余弦定理求 θ₄

```
cos(θ₄) = (|SW|² - l_se² - l_ew²) / (2·l_se·l_ew)
```

θ₄ 只由 |SW| 决定，与臂角 ψ 无关。

### Step 3：臂角 ψ → 肘位置 E

肘 E 被约束在一个圆上：圆心 C 在 SW 连线上，ψ 是 E 在圆上的转角。

```
x  = (l_se² - l_ew² + l_sw²) / (2·l_sw)    # 圆心到 S 的投影长度
r  = sqrt(l_se² - x²)                        # 圆半径
C  = S + x·u_sw                              # 圆心

e1 = normalize(S × u_sw)                     # 圆平面基向量1
e2 = u_sw × e1                               # 圆平面基向量2

E  = C + r·(cos(ψ)·e1 + sin(ψ)·e2)          # 肘位置
```

### Step 4：从 E、W 坐标直接解 θ₁θ₂θ₃

利用 FK 位置方程反解（对 s2 取正负，产生两组解）：

```
q2: c2 = (Ez - d0) / d2            # 从肘的 Z 坐标读出
q1: c1 = -Ex/(d2·s2),  s1 = -Ey/(d2·s2)
q3: 用 E→W 方向向量与 FK 旋转列对齐求解
```

### Step 5：从 T₄₇ 解腕部 θ₅θ₆θ₇

```
T47 = T_0_4⁻¹ · T_target
c6 = T47[1,2]
θ₅、θ₆、θ₇ 由 atan2 直接从矩阵元素读出（sgn = ±1，产生两组）
```

### Step 6：选最优臂角

- 暴力扫 [-π, π)，步长 0.01（约 628 次求解）
- 代价函数：关节限位加权惩罚（指数型）× 角度变化量²

## cpp 版求解流程（代数路线）

### Step 1：腕心向量

```
l_7_wt = sqrt(d6² + d7²)                        # 合并两段偏置
x_0_sw = P_desire - l_0_bs - R_desire·l_7_wt   # 一步得到 SW 向量
```

### Step 2：余弦定理求 θ₄

```
cos(θ₄) = (|SW|² - l_se² - l_ew²) / (2·l_se·l_ew)
θ₄ = -acos(cos_θ₄)      # 负号为 DH 约定 offset
```

### Step 3：构造 A_s, B_s, C_s 矩阵

```
R_0_3_ref = Rz(θ₁_ref)·Ry(-θ₂_ref)·Rz(0)    # 臂角=0 时的参考姿态
ux = skew(u_sw)                                 # u_sw 反对称矩阵

A_s =  ux · R_0_3_ref
B_s = -ux · ux · R_0_3_ref
C_s =  (u_sw · u_swᵀ) · R_0_3_ref
```

### Step 4：臂角 ψ → R_0_3

```
R_0_3 = A_s·sin(ψ) + B_s·cos(ψ) + C_s        # Rodrigues 参数化
```

### Step 5：ZYZ 分解求 θ₁θ₂θ₃

```
θ₂ = atan2(sqrt(R(2,0)² + R(2,1)²), R(2,2))
θ₁ = atan2(-R(1,2), -R(0,2))
θ₃ = atan2(-R(2,1),  R(2,0))
```

### Step 6：从 R₄₇ 解腕部 θ₅θ₆θ₇

```
R_4_7 = R_3_4ᵀ · R_0_3ᵀ · R_desire
θ₅、θ₆、θ₇ 由同样的 ZYZ 分解读出
```

### Step 7：选最优臂角

- 在当前臂角附近均匀采 **15 点**，或黄金分割搜索
- 代价函数：关节角变化² + 速度变化（加速度代理）+ 奇异性钟形惩罚

---

## 关键差异汇总

| 步骤 | py 版 | cpp 版 |
|---|---|---|
| 腕心偏置 | 两段分别退（d8 + d6） | 合并为一个向量 sqrt(d6²+d7²) |
| θ₄ 符号 | +acos（正值） | -acos（DH offset） |
| 核心参数化对象 | 肘**位置** E | 肩部**旋转矩阵** R_0_3 |
| q1~q3 求法 | FK 位置方程直接代入 | ZYZ 矩阵分解 |
| 腕部 T47 元素 | T47[1,2] = c6 | R_4_7 第三列/行 |
| ψ 搜索范围 | 全 [-π, π)，628 次 | ±7 点或黄金分割，≤20 次 |
| 代价函数 | 限位加权指数惩罚 | dq² + 加速度代理 + 奇异惩罚 |

---

## 小结

两者都属于**臂角参数化**的解析逆解框架，θ₄ 求法完全等价。核心分歧在于：

- **py 版**：ψ → 肘位置 E → 代入 FK 位置方程。直觉清晰，但暴力搜索开销大。
- **cpp 版**：ψ → Rodrigues 旋转 → R_0_3 矩阵分解。计算更紧凑，搜索策略适合实时。
